## Run Locally

Clone the project

```bash
  git clone https://github.com/pramodmahato/PhonePe.git
```

Go to the project directory

```bash
  cd PhonePe
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
